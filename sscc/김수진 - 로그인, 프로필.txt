<로그인>
1. 로그인 선택 
2. 로그인 화면 
3. 이메일 비번 입력
4. 이메일과 비번이 db와 일치하는지 확인
5. 있는 경우 로그인 상태
6-1. (개인회원)로그아웃, 회원정보 조회/수정 버튼 표시
6-2. (중개사 회원) 로그아웃, 회원정보 조회/수정, 프로필 조회수정 버튼 표시
7. 로그아웃 전까지 로그인 유지

4-1. 아이디가 회원목록에 없는 경우 
1). 아이디가 틀렸다는 팝업창을 띄우고
확인창을 누르면 입력창에 커서 위치

<회원가입>
1. 회원정보를 입력한다
2. 회원가입 버튼을 누르면 db에 가입 정보가 입력되면서 회원가입이 완료되었다는 팝업창
확인을 누르면 로그인 화면으로 이동
2-1. 누락된 항목이 있으면 팝업창으로 알리면서 해당 창에 커서 위치
3. 취소 버튼을 누르면 로그인 화면으로 이동

<회원정보>
1. 회원정보 조회 버튼을 클릭하면 회원정보 페이지로 이동
2. 회원정보 수정 버튼을 클릭하면 회원정보 수정 페이지로 이동
3. 변경 할 항목을 수정 후 비밀번호를 2회 동일하게 입력 후 수정 버튼 클릭
4. 비밀번호 2개가 일치 하고 현재 로그인 중인 아이디의 db상 비밀번호와 일치하면 db 정보 수정 후 회원정보가 완료되었습니다 팝업창 표시
확인 창 누르면 회원정보 페이지로 이동
4-1. 비밀번호 2개가 일치하지 않을 시 비밀번호를 동일하게 입력하여 주십시오 팝업창

<프로필 조회/수정>
1. 프로필 조회 버튼을 클릭
2. 프로필 페이지로 이동
3. 수정 버튼을 클릭하면 프로필 수정 페이지로 이동
4. 프로필 수정 후 비밀번호를 입력하면 프로필 변경 완료 팝업 창이 뜸 확인버튼을 누르면 프로필 조회 페이지로 이동
4-1. 비밀번호가 일치하지 않으면 비밀번호를 다시 입력하여 주십시오 팝업창을 표시 후 확인버튼을 클릭하면 기존 비밀번호 삭제 후 커서 위치시킴
5. 취소 버튼을 클릭하면 프로필 조회 페이지로 이동
